## 🚀 Instalando o Todo App - PWI

Para instalar o Todo App, siga estas etapas:

Linux e macOS:

```
git clone https://github.com/pedrocmoreira/todo-app-pwi.git
cd todo-app-pwi
npm install
npx expo start
(escolher a opção desejada para iniciar a aplicação - i para iOS / a para Android)
```

Windows:

```
git clone https://github.com/pedrocmoreira/todo-app-pwi.git
cd todo-app-pwi
npm install
npx expo start
selecionar a para iniciar a aplicação no android
```
